# 🏨 Hotel Management API

A backend application developed while learning **Node.js**, **Express.js**, and **MongoDB**. This project demonstrates RESTful API development, authentication, database integration, and modular backend architecture by implementing a hotel management system.

---

## 🚀 Features

- Employee Authentication
- Role-Based Access
- CRUD Operations
- Hotel Staff Management
- Menu Management
- Customer Data Management
- MongoDB Integration
- RESTful APIs
- Modular MVC Architecture

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Passport.js
- bcrypt
- REST APIs

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/Shashwat-pandey21/Hotel-Management-API.git
```

Install dependencies

```bash
npm install
```

Start MongoDB

```bash
mongod
```

Run the server

```bash
npm start
```

Server runs on

```
http://localhost:3000
```
